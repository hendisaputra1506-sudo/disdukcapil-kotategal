# hooks/

Folder ini berisi **custom React hooks** yang dapat digunakan ulang lintas halaman dan komponen.

## Konvensi Penamaan
- Semua file hooks diawali dengan `use`, contoh: `useScrollTop.js`, `useMediaQuery.js`
- Setiap hook dalam file terpisah

## Contoh hooks yang akan dibuat
| File | Kegunaan |
|---|---|
| `useScrollTop.js` | Auto scroll ke atas saat navigasi halaman |
| `useMediaQuery.js` | Deteksi breakpoint responsive |
| `useClickOutside.js` | Deteksi klik di luar elemen (misal dropdown menu) |
