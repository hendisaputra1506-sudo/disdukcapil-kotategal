# components/ui/

Folder ini berisi **komponen UI atom dan molekul** yang bersifat reusable dan tidak terikat pada halaman tertentu.

## Konvensi
- Setiap komponen dalam file terpisah (PascalCase)
- Props harus terdokumentasi dengan JSDoc
- Tidak boleh mengandung logika bisnis — hanya presentasi

## Contoh komponen yang akan dibuat
| Komponen | Kegunaan |
|---|---|
| `Badge.jsx` | Label kategori/tag pada berita |
| `Card.jsx` | Container konten berita/layanan |
| `Button.jsx` | Tombol dengan varian (primary, outline, ghost) |
| `SectionTitle.jsx` | Judul bagian halaman yang konsisten |
